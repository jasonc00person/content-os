# Winning Hooks

## Hook Mechanics To Reuse

### "Are you still..."

Best for calling out manual behavior that should feel outdated.

Examples to generate from:
- "Are you still posting manually from the app in 2026?"
- "Are you still writing captions from scratch?"
- "Are you still guessing what to post?"

Why it fits Jason: it is direct, slightly confrontational, and sets up an AI-system demo.

### "Comment X to get..."

Best for resource reels. Strongest Instagram CTA pattern in the current reports.

Use when the asset is concrete:
- a skill
- a prompt pack
- a content system folder
- a checklist
- a teardown

Weak version: "comment for more info." Strong version: "Comment SKILL and I'll send you the exact Codex skill that posts for me."

### "I just built..."

Best for build-in-public and system demos.

Examples:
- "I just built a Codex skill that researches my competitors for me."
- "I just built an AI employee that posts my reels."
- "I just built a content team inside a folder."

This should usually show the system within the first few seconds.

### "Everyone is using AI for X. Nobody is building Y."

Best for anti-LARP positioning.

Source pattern: the "Claude + Nano Banana" winner says everyone uses AI to create content, nobody builds the system behind it.

Jason version:
- "Everyone is using AI to write captions. Nobody is building the content machine."
- "Everyone is prompting ChatGPT. Nobody is building actual AI employees."

### "If I were starting over..."

Best for YouTube long-form or slower strategy reels.

Use sparingly. This is proven, but crowded. Needs a concrete twist:
- "If I were starting content from zero in 2026, I'd build this folder first."
- "If I had 200 views and no editor, this is the AI system I'd install."

### Time-bound challenge

Best for long-form or high-retention demos.

Examples:
- "Watch me make a viral reel in 1 hour."
- "Can Codex take this from idea to scheduled post in 60 minutes?"
- "I gave myself one hour to build an AI content team."

## Jason-Specific Hook Rules

- Lead with the spoken line, not only the text overlay.
- Make the viewer feel behind if they are still doing the manual workflow.
- Show proof or the live system fast.
- Tie hooks back to Skool Standard when the viewer is broad; tie to Inner Circle only when the viewer is established and stuck.
