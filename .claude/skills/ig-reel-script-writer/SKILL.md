# Instagram Reel Script Writer

Transform raw ideas, rambles, and rough concepts into viral Instagram Reel scripts optimized for retention, engagement, and conversions — for any creator, brand, or niche.

This skill is built on a specific philosophy of content: substance over flash, extreme value delivery, subtle psychology, and showing rather than telling. The methodology works across every niche because the underlying psychology of attention and persuasion is universal. What changes between a fitness creator and a real estate agent is the *subject matter* — not the structure, not the hooks, not the persuasion principles.

## Workflow

Writing a reel script involves these steps:

1. Understand who the script is for and clarify the idea
2. Read the scripting methodology (`references/scripting-methodology.md`)
3. Select a hook format (consult `references/viral-hooks.md` for proven formats)
4. Draft the full script using the template (`references/script-template.md`)
5. Annotate each section with the psychology tricks being used
6. Write an Instagram caption with hook, value summary, and CTA

## Step 1: Understand the Creator & Clarify the Idea

Before writing anything, you need to understand two things: **who is the creator** and **what is the reel about**.

### Who is this script for?

If context about the creator/brand hasn't been provided yet, figure out:

- **Who is the creator?** — Name, brand, what they do, what they're known for
- **What's their niche/industry?** — Fitness, business, tech, real estate, beauty, education, etc.
- **Who is THEIR audience?** — Demographics, pain points, aspirations, sophistication level
- **What's their voice/tone?** — Casual and funny? Authoritative and data-driven? Raw and unfiltered? Polished and professional?
- **What's their authority/credibility?** — Results they've achieved, social proof they can reference, credentials

If the user has already provided this context (e.g., they mentioned the client, shared notes, or you've written scripts for this person before in the conversation), don't re-ask. Use what you have.

If the user IS the creator (writing for themselves), that's fine too — just understand their niche, audience, and voice.

### What is this reel about?

When the user provides an idea (which may be a rough ramble, a topic, or a polished concept), extract:

- **Core topic/message** — What is this reel actually about?
- **Goal** — Views (mass appeal hook) or conversions (specific/tactical hook)?
- **Duration** — 15s, 30s, 60s, or 90s? (Default: 60s)
- **Content type** — Quick tips, tutorial, story-based, or "moments"?

If the user just rambles an idea, distill it into a clear concept. Do NOT ask excessive clarifying questions — make smart defaults based on the creator's niche and audience, then write the script. The user can iterate.

## Step 2: Read the Methodology

Before writing, read `references/scripting-methodology.md` to load the full viral scripting framework including:
- The 5-part structure (Hook → Build-Up → Value → Payoff → CTA)
- The 6 principles of persuasion
- The psychology tricks toolkit
- Key principles (subtlety, authority through showing, every line must serve a purpose)

## Step 3: Select a Hook

Read `references/viral-hooks.md` to find a proven hook format that fits the topic. The hook library contains:
- 100+ real hooks with view counts (sorted by virality)
- Hook format categories (Dream Outcome, Burning Question, Challenge, Controversy, etc.)
- Fill-in-the-blank hook templates

**Hook selection rules:**
- If goal is VIEWS → pick a mass appeal hook format
- If goal is CONVERSIONS → can be more specific/tactical
- Always adapt proven formats to the creator's topic rather than inventing from scratch
- The hooks in the library span many niches — the *format* is what matters, not the specific niche of the original. A "Day in the life of a TikTok clipper making $20k/month" hook works just as well reframed as "Day in the life of a realtor closing $2M in deals a month"
- Present 2-3 hook options to the user when possible

## Step 4: Write the Script

Read `references/script-template.md` for the exact output format and duration guidelines. Then write the full script following the 5-part structure.

**Critical writing rules:**
- Write in the creator's voice — match their tone, vocabulary, and energy level. A Gen Z fitness creator sounds nothing like a 45-year-old financial advisor, and the script needs to feel authentic to whoever is saying it on camera.
- Every single line must serve a purpose (annotated with its psychology trick)
- Cut ALL fluff — if a line isn't doing psychological work, remove it
- Be subtle with psychology tricks — they should be invisible to the viewer
- Show authority through setting/proof, not by bragging — adapt this to the creator's actual credibility (their results, their experience, their clients)
- Hold the payoff as long as possible — everything between hook and payoff drives watch time
- Value must be SPECIFIC and ACTIONABLE — "Would the viewer pay $5 for this?"
- Give away the best stuff — paid-level value for free
- CTA should leverage reciprocity — prefer "keep giving" CTAs over "please follow" CTAs
- The CTA should make sense for the creator's business model (e.g., if they sell courses, the CTA might point to a free resource; if they're a service provider, it might be a free consultation)

## Step 5: Deliver the Script

Output the script in the annotated template format (from `references/script-template.md`), including:
- Target length and hook type
- Psychology tricks used
- Each section with inline annotations explaining WHY each line exists
- A ready-to-use Instagram caption with hashtags relevant to the creator's niche

## Iteration

After delivering, the user may want to:
- Try different hook angles → re-consult the hook library
- Adjust the tone (more casual, more authority, more controversy)
- Change the duration
- Swap the CTA strategy
- Add more extreme value moments
- Rewrite for a different audience segment

Always be ready to quickly iterate on any section.
