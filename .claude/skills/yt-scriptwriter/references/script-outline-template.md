# Script Outline Template

Use this template to structure every YouTube script before writing the full prose. Fill in each section, then use it as the skeleton for the complete script.

---

## Pre-Script Setup

```
VIDEO TITLE: [Working title — this is what the hook will confirm]
THUMBNAIL CONCEPT: [Brief description of the thumbnail visual]
TARGET LENGTH: [5-8 min / 10-15 min / 20+ min]
GOAL: [Education / Entertainment / Story / Hybrid]
CTA DESTINATION: [Next video title or action]

--- 3 PRE-SCRIPT QUESTIONS ---

500 SUBS (Beginner appeal):
[How does this help someone just starting out? What frameworks/examples make this actionable for a beginner?]

10M SUBS (Expert appeal):
[What's new or fresh here that even an advanced person hasn't seen? New data, formats, angles?]

MOM (Outsider appeal):
[What universal element (story, emotion, data, curiosity) makes this interesting to someone who knows nothing about the topic?]
```

## Script Structure

### HOOK

```
AUDITORY (what you say — 1-3 lines):
[Slight variation of the title. Confirm the promise.]

VISUAL (what the viewer sees):
[Match the thumbnail. Describe the opening frame, b-roll, animation, or on-camera setup.]
```

### INTRO (30-60 seconds)

```
BEAT 1 — Confirm title/thumbnail:
[Hook line, transitioning into intro]

BEAT 2 — Social proof #1:
[Result, credential, or client win that establishes authority]

BEAT 3 — Open loop / new narrative:
[Raise a question that won't be answered yet. Create incompleteness.]

BEAT 4 — Set expectations + Social proof #2:
[What the viewer will get from this video + another credibility line]

BEAT 5 — Singularity:
[Why is YOUR video/perspective unique? What's your mission or POV?]
```

### BODY

For each major section of the video, fill in this block:

```
--- SECTION [#]: [Topic/Title] ---

QUESTIONS TO ANSWER (for assumptive questions technique):
- [Question 1 the viewer would have]
- [Question 2]
- [Question 3]
(You'll answer these in the script and DELETE the questions)

KEY POINTS:
- [Main point 1]
- [Main point 2]
- [Main point 3]

OPEN LOOP (at the end of this section):
[What question do you raise to keep them watching into the next section?]

PAYOFF (at the start of this section — closing a previous loop):
[What previous question are you answering here?]

EMOTIONAL DIRECTION:
[LOW → HIGH / HIGH → LOW / Neutral → Build]
[What struggle/pain do you touch on? What transformation/energy do you bring?]
```

### OPEN LOOP / PAYOFF MAP

Map out the cycling pattern for the full body:

```
Section 1: Payoff (intro loop closed) → Open Loop A
Section 2: Payoff (Loop A closed) → Open Loop B
Section 3: Payoff (Loop B closed) → Open Loop C
Section 4: Payoff (Loop C closed) → Open Loop D
...
Climax: Biggest payoff + biggest emotional swing
```

### CLIMAX

```
CLIMAX TYPE: [Biggest high → lowest low / Lowest low → biggest high / Plot twist / Revelation]

SETUP (what leads into the climax):
[How are you building tension/stakes right before this moment?]

THE MOMENT:
[What happens at the peak? What does the viewer feel?]

AFTERMATH:
[How do you bring the viewer down from the climax? Falling action / reflection / resolution]
```

### CTA

```
LINK (bridge from current topic):
[Connect naturally from whatever you just finished discussing]

CURIOSITY GAP (one final open loop):
[Raise a question about the next video/action that creates incompleteness]

CTA + PROMISE:
[Tell them what to do + what they'll get]
"Check out this video where I show you [X] and how you can use it to [specific result]."

END SCREEN:
[Point to the video. Use the card.]
```

## Story Structure Selection

Choose the macro structure for the body based on your video type:

### For Entertainment / Challenge Videos → Three-Act Structure
```
Act 1 (Setup): Introduce characters, stakes, foreshadow. End with a plot point.
Act 2 (Confrontation): Rising action. Stakes escalate. Faster cuts, more tension.
Act 3 (Climax + Falling Action): Peak moment → emotional release → resolution.
```

### For Transformation / Adventure Videos → Hero's Journey
```
1. Status Quo → 2. Call to Adventure → 3. Assistance → 4. Departure →
5. Trials → 6. Approach → 7. Crisis/Climax → 8. Treasure/Result →
9. Return → 10. New Life → 11. Resolution
```

### For Educational / Tutorial Videos → Open Loop Flow
```
Intro → Payoff/Loop cycle → Payoff/Loop cycle → ... → Climax (biggest insight) → CTA
Use the emotional rollercoaster (lows + highs) between sections.
Connect every point with the river rocks principle.
```

## Checklist Before Writing

Before converting this outline into a full script, verify:

- [ ] Title is confirmed in the hook (auditory + visual)
- [ ] At least 2-3 social proof moments mapped in the intro
- [ ] Every section has an open loop AND a payoff mapped
- [ ] Emotional highs and lows are planned (not all neutral)
- [ ] Climax is clearly identified and set up
- [ ] CTA uses the 3-part framework (Link → Gap → Promise)
- [ ] The 3 pre-script questions are answered and shaping the content
- [ ] Questions are written in the outline (to be answered and deleted in the script)
- [ ] Every section connects logically to the one before it
