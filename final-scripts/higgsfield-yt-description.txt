In this video, I break down my exact workflow for creating super cinematic shots of yourself using Higgsfield AI — perfect for Instagram reels, YouTube B-roll, or any content you're making.

What's covered:
- Creating your AI character (10-15 photos is all you need)
- Generating images with the right models (Soul Cinema vs. Nano Banana Pro)
- Turning images into video using Kling 3.0
- Post-production: color grading + film grain to make it look legit

The Workflow:
1. Create Character → upload 10-15 photos of yourself
2. Generate Image → use Nano Banana Pro or Soul Cinema, batch size of 4, run multiple generations
3. Create Video → upload your best image as the start frame, add a motion prompt
4. Edit → throw on a LUT + film grain overlay in your editor

Pro Tips:
- Start with an image FIRST, then turn it into video (don't go straight to video)
- Use Gemini to describe reference images, then plug that description into your prompt
- Run 10-20+ generations to find the best one — don't settle for the first result
- Color grade + film grain makes everything look 10x more cinematic
