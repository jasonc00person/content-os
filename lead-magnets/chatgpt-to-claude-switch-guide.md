## ⚡ Step 3 — Install Claude Code (The Real Unlock)

This is the move most people skip. Don't.

**Claude Code** is a terminal tool that lets Claude run your computer. It can:

- ✅ Read and edit any file on your machine
- ✅ Run shell commands, scripts, and tests
- ✅ Control your browser (automate DMs, scrape data, post content)
- ✅ Build entire applications from a single prompt

### Install It

Open your terminal and run:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Then authenticate:


It'll pop open a browser, log you in, done.

## 🎯 Step 4 — Prompting Differences (Adjust Your Style)

ChatGPT trained you to over-explain, hedge, and add "please" to everything. Claude doesn't need that.

### What Works Better with Claude

| Do This | Not This |
|---------|---------|
| Be direct. "Write X. Do Y." | "Could you possibly help me with..." |
| Give context once, then iterate | Re-explain everything every message |
| Let it ask questions | Dump a wall of instructions |
| Trust its judgment | Micromanage every sentence |
| Say "make it better" and it will | Try to prompt-engineer perfection |

### Prompting Formula I Use

```
ROLE: You're a [specific expert].
CONTEXT: [situation, audience, constraints]
TASK: [what you want]
OUTPUT: [format — bullets, script, code, table]
```

Example:

> You're a short-form video scriptwriter. I coach business owners doing $10-40k/mo who need AI systems. Write me 5 hooks for a reel about why ChatGPT is slowing them down. Casual, punchy, contrarian. 15 words max each.

Claude nails it first try.
